# Тестовое задание — Vue 3 / Pinia / Tailwind

Приложение для отображения постов и пользователей с сервиса JSONPlaceholder.

## Технологии
- Vue 3 (Composition API, `<script setup>`)
- Pinia (кэш, состояния)
- Tailwind CSS
- JSONPlaceholder API
- lucide-vue-next (иконки)

## Требования по ТЗ
- Загрузка постов и пользователей при старте приложения
- Таблица 600×600, по центру, скролл только внутри
- Прилипшая шапка таблицы
- Поиск по `title`
- Если поле поиска пустое → показывать все посты
- Подгрузка по 30 элементов при прокрутке (infinite scroll)
- Сортировка по всем колонкам (по клику, ASC/DESC)
- Tooltip через `title`, если текст обрезан
- Модалка пользователя:
  - name, username, email, phone, website, company.name, полный адрес
  - закрытие по кнопке и по фону
- Данные модалки берутся из кэша
- Просмотренные пользователи сохраняются в `localStorage`
- Переключатель тем (light/dark), хранение темы в `localStorage`
- Скелетон/индикатор загрузки
- Обработка ошибок API (toast)

## Реализовано
- Полная загрузка данных при старте (users + posts)
- Кэширование в Pinia
- Поиск + автоматический сброс при очистке поля
- Infinite scroll через `IntersectionObserver`
- Полная сортировка таблицы
- Скелетон при первичной загрузке
- Tooltip через `title`
- Модальное окно с данными пользователя
- Закрытие модалки по клику на фон/кнопку
- Метка "просмотрено" для email (сохранение в localStorage)
- Переключение темы + сохранение в localStorage
- Адаптивная верстка
- Аккуратные стили под светлую и тёмную тему
